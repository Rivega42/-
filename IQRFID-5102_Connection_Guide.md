# IQRFID-5102 Считыватель - Инструкция по подключению

## Технические параметры

- **Baud Rate**: 57600
- **Data Bits**: 8
- **Stop Bits**: 1
- **Parity**: None
- **Flow Control**: None

## Протокол связи

### Формат фрейма
```
[LEN][ADR][CMD][DATA...][CRC_LOW][CRC_HIGH]
```

- **LEN** - длина данных после байта длины (не включает сам байт LEN)
- **ADR** - адрес устройства (обычно `0x00`)
- **CMD** - код команды
- **DATA** - данные команды (если есть)
- **CRC** - контрольная сумма CRC-16 (полином 0x8408, LSB first)

### Команда Inventory (чтение тегов)

**Отправка:**
```
04 00 01 DB 4B
```
- `04` - длина (4 байта после этого байта)
- `00` - адрес
- `01` - команда Inventory
- `DB 4B` - CRC

### Ответы

**Нет тегов:**
```
05 00 01 FB F2 3D
```
- `05` - длина
- `00` - адрес
- `01` - команда
- `FB` - статус "нет тегов"
- `F2 3D` - CRC

**Тег найден:**
```
13 00 01 01 01 0C [EPC_DATA] [CRC]
```
- `13` - длина (19 байт)
- `00` - адрес
- `01` - команда
- `01` - количество тегов
- `01` - RSSI
- `0C` - длина EPC (12 байт)
- `[EPC_DATA]` - данные EPC (12 байт)
- `[CRC]` - контрольная сумма (2 байта)

## Пример реального ответа

```
13 00 01 01 01 0C 30 4D B7 5F 19 60 00 21 F0 13 82 50 29 BA
```

**Декодирование:**
- Длина: 19 байт (0x13)
- Адрес: 0x00
- Команда: 0x01 (Inventory)
- Количество тегов: 1
- RSSI: 1
- Длина EPC: 12 байт (0x0C)
- **EPC: `304DB75F19600021F0138250`**
- CRC: 0x29BA

## Алгоритм CRC-16

```
Polynomial: 0x8408
Initial Value: 0xFFFF
LSB First (Little Endian)
```

## Подключение в приложении

1. Выберите COM порт считывателя
2. Выберите тип "IQRFID-5102"
3. Нажмите "Connect"
4. Поднесите RFID карту - считыватель пикнет и отобразит EPC
