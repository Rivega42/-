# Операции с полочками BookCabinet

## Общая концепция

Шкаф имеет **два ряда** полочек:
- **Передний ряд (depth=1)** — ближе к пользователю, работает только передний замок
- **Задний ряд (depth=2)** — дальше, требует перехват между замками

**Точка перехвата** — позиция, где замок находится на середине каретки и может зацепить/отпустить полочку.

---

## Задний ряд (depth=2)

### Выдвижение полочки

```
1. Платформа → BACK концевик
2. Задний замок → 1200 (захват) + pigs s 13 0
3. Платформа → точка перехвата заднего замка (~17000 шагов к FRONT)
4. Задний замок → 500 (отпустить) + pigs s 13 0
5. Платформа → точка перехвата переднего замка (+12800 к BACK)
6. Передний замок → 1200 (захват) + pigs s 12 0
7. Платформа → полный выезд (~18800 к FRONT, полочка на каретке)
```

### Возврат полочки

```
1. Передний замок → 1200 (держит полочку) + pigs s 12 0
2. Платформа → точка перехвата переднего замка (+12800 к BACK)
3. Передний замок → 500 (отпустить) + pigs s 12 0
4. Платформа → точка перехвата заднего замка (+12800 к FRONT)
5. Задний замок → 1200 (захват) + pigs s 13 0
6. Платформа → BACK концевик
7. Задний замок → 500 (отпустить) + pigs s 13 0
8. Платформа → центр
```

## Передний ряд (depth=1)

Для переднего ряда **не нужен перехват** между замками — работает только передний замок.

### Выдвижение полочки

**Начальное состояние:** полочка в ячейке переднего ряда, платформа в центре

```
Шаг 1: Платформа → FRONT концевик
        - Едем DIR=0 до срабатывания FRONT (GPIO 7)
        - Платформа подъехала к ячейке

Шаг 2: Передний замок → захват
        - pigs s 12 1200
        - sleep 0.5
        - pigs s 12 0  ← отключить PWM!
        - Замок зацепил полочку в ячейке

Шаг 3: Платформа → назад на ~4500 шагов (к BACK)
        - Едем DIR=1 на 4500 шагов
        - Тянем полочку на каретку
        - Передний замок теперь на середине каретки

Шаг 4: Передний замок → отпустить
        - pigs s 12 500
        - sleep 0.5
        - pigs s 12 0  ← отключить PWM!
        - Полочка лежит на каретке, готова к работе
```

**Конечное состояние:** полочка на каретке, замок открыт

### Возврат полочки

**Начальное состояние:** полочка на каретке

```
Шаг 1: Передний замок → захват
        - pigs s 12 1200
        - sleep 0.5
        - pigs s 12 0  ← отключить PWM!
        - Замок держит полочку

Шаг 2: Платформа → FRONT концевик
        - Едем DIR=0 до срабатывания FRONT (GPIO 7)
        - Толкаем полочку в ячейку

Шаг 3: Передний замок → отпустить
        - pigs s 12 500
        - sleep 0.5
        - pigs s 12 0  ← отключить PWM!
        - Полочка остаётся в ячейке

Шаг 4: Платформа → центр
        - Едем DIR=1 на TRAY_CENTER (~10750) шагов
        - Платформа в исходной позиции
```

**Конечное состояние:** полочка в ячейке, платформа в центре

### Особенности переднего ряда

- **Проще заднего** — нет перехвата между замками
- **Быстрее** — меньше шагов
- **Задний замок НЕ используется** — можно не инициализировать

## Перемещение между рядами

### Из переднего ряда в задний

**Начальное состояние:** полочка в передней ячейке, платформа в центре

```
=== ЧАСТЬ 1: ВЗЯТЬ ИЗ ПЕРЕДНЕГО РЯДА ===

Шаг 1: Платформа → FRONT концевик
        - Едем DIR=0 до FRONT (GPIO 7)

Шаг 2: Передний замок → захват
        - pigs s 12 1200 && sleep 0.5 && pigs s 12 0

Шаг 3: Платформа → точка перехвата переднего замка
        - Едем DIR=1 на ~4500 шагов
        - Полочка на каретке, передний замок на середине

=== ЧАСТЬ 2: ПЕРЕДАТЬ ЗАДНЕМУ ЗАМКУ ===

Шаг 4: Платформа → точка перехвата заднего замка
        - Едем DIR=1 ещё на +12800 шагов (расстояние между замками)
        - Задний замок теперь на середине каретки

Шаг 5: Задний замок → захват
        - pigs s 13 1200 && sleep 0.5 && pigs s 13 0
        - Задний замок держит полочку

Шаг 6: Передний замок → отпустить
        - pigs s 12 500 && sleep 0.5 && pigs s 12 0
        - Передали полочку заднему замку

=== ЧАСТЬ 3: ЗАСУНУТЬ В ЗАДНИЙ РЯД ===

Шаг 7: Платформа → BACK концевик
        - Едем DIR=1 до BACK (GPIO 20)
        - Толкаем полочку в заднюю ячейку

Шаг 8: Задний замок → отпустить
        - pigs s 13 500 && sleep 0.5 && pigs s 13 0
        - Полочка в задней ячейке

Шаг 9: Платформа → центр
        - Едем DIR=0 на TRAY_CENTER (~10750) шагов
```

**Конечное состояние:** полочка в задней ячейке, платформа в центре

---

### Из заднего ряда в передний

**Начальное состояние:** полочка в задней ячейке, платформа в центре

```
=== ЧАСТЬ 1: ВЗЯТЬ ИЗ ЗАДНЕГО РЯДА ===

Шаг 1: Платформа → BACK концевик
        - Едем DIR=1 до BACK (GPIO 20)

Шаг 2: Задний замок → захват
        - pigs s 13 1200 && sleep 0.5 && pigs s 13 0

Шаг 3: Платформа → точка перехвата заднего замка
        - Едем DIR=0 на ~17000 шагов
        - Полочка на каретке, задний замок на середине

=== ЧАСТЬ 2: ПЕРЕДАТЬ ПЕРЕДНЕМУ ЗАМКУ ===

Шаг 4: Платформа → точка перехвата переднего замка
        - Едем DIR=1 на +12800 шагов
        - Передний замок теперь на середине каретки

Шаг 5: Передний замок → захват
        - pigs s 12 1200 && sleep 0.5 && pigs s 12 0
        - Передний замок держит полочку

Шаг 6: Задний замок → отпустить
        - pigs s 13 500 && sleep 0.5 && pigs s 13 0
        - Передали полочку переднему замку

=== ЧАСТЬ 3: ЗАСУНУТЬ В ПЕРЕДНИЙ РЯД ===

Шаг 7: Платформа → FRONT концевик
        - Едем DIR=0 до FRONT (GPIO 7)
        - Толкаем полочку в переднюю ячейку

Шаг 8: Передний замок → отпустить
```

## Константы

```python
# Платформа
TRAY_TOTAL = 21500
TRAY_CENTER = 10750
LOCK_DISTANCE = 12800      # Расстояние между замками

# Точки перехвата (от соответствующего концевика)
HANDOFF_FROM_FRONT = 4500  # Передний замок на середине каретки
HANDOFF_FROM_BACK = 4500   # Задний замок на середине каретки

# Полный выезд (от точки перехвата к FRONT)
FULL_EXTEND = 18800        # Шагов к FRONT чтобы полочка легла на каретку

# Замки
LOCK_GRAB = 1200           # Угол захвата (~70°)
LOCK_RELEASE = 500         # Угол отпускания (0°)

# GPIO
LOCK_FRONT = 12
LOCK_BACK = 13
```

---

## Важные правила

1. **После каждой команды серво** — отключать PWM:
   ```bash
   pigs s 13 1200 && sleep 0.5 && pigs s 13 0
   ```

2. **Не ехать в нажатый концевик** — проверять перед движением

3. **Калибровка только при хоминге** — не каждый цикл работы с полочкой

4. **Использовать wave_chain** — не wave_send_once в цикле!

## Полный цикл (пример кода)

```python
def lock_grab(pin):
    """Захватить замком"""
    os.system(f"pigs s {pin} 1200")
    time.sleep(0.5)
    os.system(f"pigs s {pin} 0")

def lock_release(pin):
    """Отпустить замок"""
    os.system(f"pigs s {pin} 500")
    time.sleep(0.5)
    os.system(f"pigs s {pin} 0")

def extract_back_shelf():
    """Выдвинуть полочку из заднего ряда"""
    tray_to_endstop(BACK)
    lock_grab(LOCK_BACK)
    tray_move(17000, direction=FRONT)
    lock_release(LOCK_BACK)
    tray_move(LOCK_DISTANCE, direction=BACK)
    lock_grab(LOCK_FRONT)
    tray_move(FULL_EXTEND, direction=FRONT)

def return_back_shelf():
    """Вернуть полочку в задний ряд"""
    lock_grab(LOCK_FRONT)
    tray_move(LOCK_DISTANCE, direction=BACK)
    lock_release(LOCK_FRONT)
    tray_move(LOCK_DISTANCE, direction=FRONT)
    lock_grab(LOCK_BACK)
    tray_to_endstop(BACK)
    lock_release(LOCK_BACK)
    tray_to_center()

def extract_front_shelf():
    """Выдвинуть полочку из переднего ряда"""
    tray_to_endstop(FRONT)
    lock_grab(LOCK_FRONT)
    tray_move(HANDOFF_FROM_FRONT, direction=BACK)
    lock_release(LOCK_FRONT)

def return_front_shelf():
    """Вернуть полочку в передний ряд"""
    lock_grab(LOCK_FRONT)
    tray_to_endstop(FRONT)
    lock_release(LOCK_FRONT)
    tray_to_center()
```
