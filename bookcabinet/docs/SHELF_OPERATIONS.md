# Операции с полочками BookCabinet

## Общая концепция

Шкаф имеет **два ряда** полочек:
- **Передний ряд (depth=1)** — ближе к пользователю
- **Задний ряд (depth=2)** — дальше от пользователя

**Точка перехвата** — позиция платформы, где замок находится на середине каретки и может зацепить/отпустить полочку.

**Важно:** Перемещение каретки (XY) НЕ влияет на позицию платформы (tray) — они независимы.

## Откалиброванные константы (21.04.2026)

```python
# Платформа
TRAY_TOTAL = 22600         # Полный ход платформы
TRAY_CENTER = 11300        # Центральная позиция

# Задний ряд — точки перехвата (от BACK концевика)
REAR_HANDOFF_REAR = 16800   # Задний замок на середине каретки
REAR_HANDOFF_FRONT = 4200   # Передний замок на середине каретки

# Передний ряд — точки перехвата (от BACK концевика)
FRONT_HANDOFF_FRONT = 5700   # Передний замок на середине каретки
FRONT_HANDOFF_REAR = 18300   # Задний замок на середине каретки

# Расстояние между точками перехвата
LOCK_DISTANCE = 12600

# Замки PWM
LOCK_GRAB = 1200           # Захват
LOCK_RELEASE = 500         # Отпустить

# GPIO
LOCK_FRONT = 12
LOCK_REAR = 13
```

## Кросс-рядные операции

### front_to_rear (из переднего ряда в задний)

Полный цикл переноса полочки из переднего ряда в задний. 
**Можно перемещать каретку (XY) между extract и return!**

```
=== EXTRACT FROM FRONT ===
python3 shelf_operations.py extract_front
Результат: полочка на каретке, задний замок держит, платформа ~5700 от BACK

=== ОПЦИОНАЛЬНО: ПЕРЕМЕЩЕНИЕ КАРЕТКИ ===
Здесь можно двигать каретку к другой ячейке.
Позиция платформы НЕ меняется!

=== RETURN TO REAR ===
1. Задний замок → отпустить
2. Платформа → 12600 к FRONT (~18300 от BACK)
3. Передний замок → захват
4. Платформа → 12600 к BACK (~5700 от BACK)
5. Передний замок → отпустить
6. Платформа → 12500 к FRONT (~18200 от BACK)
7. Задний замок → захват
8. Платформа → BACK концевик
9. Задний замок → отпустить
10. Платформа → центр (11300 к FRONT)

Результат: полочка в ячейке заднего ряда, платформа в центре
```

## Задний ряд (depth=2)

### extract_rear — Извлечение полочки

```
Начало: полочка в ячейке, платформа в центре

1. Платформа → BACK концевик
2. Задний замок → захват
3. Платформа → 16800 к FRONT
4. Задний замок → отпустить
5. Платформа → 12600 к BACK
6. Передний замок → захват
7. Платформа → 12600 к FRONT

Конец: полочка на каретке, передний замок держит, платформа ~16800 от BACK
```

### return_rear — Возврат полочки

```
Начало: полочка на каретке, передний замок держит, платформа ~16800 от BACK

1. Платформа → 12600 к BACK
2. Передний замок → отпустить
3. Платформа → 12600 к FRONT
4. Задний замок → захват
5. Платформа → BACK концевик
6. Задний замок → отпустить
7. Платформа → центр

Конец: полочка в ячейке, платформа в центре
```

## Важные правила

1. **После каждой команды серво** — отключать PWM:
   ```bash
   pigs s 13 1200 && sleep 0.5 && pigs s 13 0
   ```

2. **Полочка на каретке — замок НЕ отпускаем** (может съехать)

3. **Калибровка платформы** — перед началом работы:
   ```bash
   python3 tray_platform.py calibrate
   ```

4. **Перемещение каретки (XY) независимо от платформы (tray)**
   - Можно двигать каретку когда полочка на платформе
   - Позиция платформы не меняется

5. **Использовать tray_platform.py** — там sensor_stable() и backoff

## GPIO карта

| Функция | GPIO | Описание |
|---------|------|----------|
| LOCK_FRONT | 12 | Передний замок |
| LOCK_REAR | 13 | Задний замок |
| TRAY_STEP | 18 | Шаг мотора платформы |
| TRAY_DIR | 27 | Направление: 0=FRONT, 1=BACK |
| TRAY_EN1 | 25 | Enable (LOW=работа) |
| TRAY_EN2 | 26 | Enable (LOW=работа) |
| ENDSTOP_FRONT | 7 | Концевик FRONT (1=нажат) |
| ENDSTOP_BACK | 20 | Концевик BACK (1=нажат) |
