# Операции с полочками BookCabinet

## Общая концепция

Шкаф имеет **два ряда** полочек:
- **Передний ряд (depth=1)** — ближе к пользователю, работает только передний замок
- **Задний ряд (depth=2)** — дальше, требует перехват между замками

**Точка перехвата** — позиция, где замок находится на середине каретки и может зацепить/отпустить полочку.

---

## Откалиброванные константы (21.04.2026)

```python
# Платформа
TRAY_TOTAL = 22650         # Полный ход платформы
TRAY_CENTER = 11325        # Центральная позиция

# Точки перехвата (от BACK концевика)
HANDOFF_REAR_FROM_BACK = 16800   # Задний замок на середине каретки
HANDOFF_FRONT_FROM_BACK = 4200   # Передний замок на середине каретки

# Расстояние между точками перехвата
LOCK_DISTANCE = 12600      # 16800 - 4200 = 12600

# Замки PWM
LOCK_GRAB = 1200           # Захват
LOCK_RELEASE = 500         # Отпустить

# GPIO
LOCK_FRONT = 12
LOCK_REAR = 13
ENDSTOP_FRONT = 7
ENDSTOP_BACK = 20
```

## Задний ряд (depth=2)

### Извлечение полочки из ячейки

**Начальное состояние:** полочка в ячейке заднего ряда, платформа в центре

```
Шаг 1: Платформа → BACK концевик
        - python3 tray_platform.py back

Шаг 2: Задний замок → захват
        - pigs s 13 1200 && sleep 0.5 && pigs s 13 0

Шаг 3: Платформа → точка перехвата заднего замка (16800 шагов к FRONT)
        - Полочка вытягивается из ячейки
        - Задний замок оказывается на середине каретки

Шаг 4: Задний замок → отпустить
        - pigs s 13 500 && sleep 0.5 && pigs s 13 0
        - Полочка лежит на каретке

Шаг 5: Платформа → точка перехвата переднего замка (12600 шагов к BACK)
        - Передний замок оказывается на середине каретки

Шаг 6: Передний замок → захват
        - pigs s 12 1200 && sleep 0.5 && pigs s 12 0
        - Передний замок держит полочку

Шаг 7: Платформа → точка перехвата заднего замка (12600 шагов к FRONT)
        - Полочка на каретке, готова к работе
```

**Конечное состояние:** полочка на каретке, передний замок держит

---

### Возврат полочки в ячейку

**Начальное состояние:** полочка на каретке, передний замок держит, платформа в точке перехвата заднего замка (16800 от BACK)

```
Шаг 1: Платформа → точка перехвата переднего замка (12600 шагов к BACK)
        - Сейчас 16800 от BACK, едем к 4200 от BACK
        - Передний замок на середине каретки

Шаг 2: Задний замок → захват
        - pigs s 13 1200 && sleep 0.5 && pigs s 13 0
        - Задний замок зацепляет полочку

Шаг 3: Передний замок → отпустить
        - pigs s 12 500 && sleep 0.5 && pigs s 12 0
        - Передали полочку заднему замку

Шаг 4: Платформа → BACK концевик
        - python3 tray_platform.py back
        - Толкаем полочку в ячейку

Шаг 5: Задний замок → отпустить
        - pigs s 13 500 && sleep 0.5 && pigs s 13 0
        - Полочка в ячейке

Шаг 6: Платформа → центр
        - Едем к FRONT на TRAY_CENTER (11325) шагов
```

**Конечное состояние:** полочка в ячейке, платформа в центре

## Передний ряд (depth=1)

Для переднего ряда **не нужен перехват** между замками.

### Извлечение полочки

```
Шаг 1: Платформа → FRONT концевик
        - python3 tray_platform.py front

Шаг 2: Передний замок → захват
        - pigs s 12 1200 && sleep 0.5 && pigs s 12 0

Шаг 3: Платформа → точка перехвата переднего замка (4200 от BACK)
        - Тянем полочку на каретку
```

**Конечное состояние:** полочка на каретке, передний замок держит

### Возврат полочки

```
Шаг 1: Платформа → FRONT концевик (передний замок держит)
        - python3 tray_platform.py front
        - Толкаем полочку в ячейку

Шаг 2: Передний замок → отпустить
        - pigs s 12 500 && sleep 0.5 && pigs s 12 0

Шаг 3: Платформа → центр
```

**Конечное состояние:** полочка в ячейке, платформа в центре

## Важные правила

1. **После каждой команды серво** — отключать PWM:
   ```bash
   pigs s 13 1200 && sleep 0.5 && pigs s 13 0
   ```

2. **Полочка на каретке — замок НЕ отпускаем** (может съехать)

3. **Калибровка платформы** — перед началом работы:
   ```bash
   python3 tray_platform.py calibrate
   ```

4. **Использовать tray_platform.py** — там sensor_stable() и backoff

## GPIO карта

| Функция | GPIO | Описание |
|---------|------|----------|
| LOCK_FRONT | 12 | Передний замок |
| LOCK_REAR | 13 | Задний замок |
| TRAY_STEP | 18 | Шаг мотора платформы |
| TRAY_DIR | 27 | Направление: 0=FRONT, 1=BACK |
| TRAY_EN1 | 25 | Enable (LOW=работа) |
| TRAY_EN2 | 26 | Enable (LOW=работа) |
| ENDSTOP_FRONT | 7 | Концевик FRONT (1=нажат) |
| ENDSTOP_BACK | 20 | Концевик BACK (1=нажат) |
