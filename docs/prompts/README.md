# Prompts

Системные промпты как код. Версионируются вместе с репо.

## Что класть

- Промпты для Claude Code по типичным задачам (bug-hunt, refactor, code-review).
- Промпты для генерации артефактов (коммиты, PR описания, changelog entries).
- System prompts для Telegram бота, kiosk UI помощника, etc.

## Конвенции

- Имя файла: `<scope>-<task>.md` (например `claude-code-bug-hunt.md`).
- В шапке — назначение и формат входа/выхода.
- В конце — пример успешного применения.
