# Guides

Howto / tutorials для контрибьюторов и операторов.

## Что класть

- `setup-dev-env.md` — как поднять dev окружение.
- `deploy-to-pi.md` — как задеплоить на Raspberry Pi.
- `calibrate-cabinet.md` — как калибровать шкаф (XY, racks, tray).
- `add-rfid-reader.md` — добавить новый тип ридера.
- `irbis-mock-setup.md` — поднять mock ИРБИС.
- `add-cabinet-instance.md` — подключить второй шкаф.

## Конвенции

- Каждый guide — самодостаточный (не надо читать 5 других файлов).
- Команды копируемые как есть, с подстановками `<...>`.
- В конце — секция "Что дальше" с ссылками.
